# Portal de Links Centralizado

Uma aplicação web leve e responsiva criada para centralizar, organizar e facilitar o acesso a sistemas e links úteis de múltiplas empresas ou setores em um único painel.

## Tecnologias Utilizadas

Este projeto foi construído focando em simplicidade e performance, utilizando as seguintes tecnologias:

**Backend:**
* [Python 3.12](https://www.python.org/) - Linguagem principal.
* [Flask](https://flask.palletsprojects.com/) - Micro-framework web para roteamento e lógica do servidor.
* [SQLite](https://www.sqlite.org/) - Banco de dados relacional embutido e leve.
* [Gunicorn](https://gunicorn.org/) - Servidor WSGI HTTP para rodar a aplicação em produção.

**Frontend:**
* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estruturação das páginas.
* [Bootstrap 5](https://getbootstrap.com/) - Framework CSS (via CDN) para interface responsiva, abas, modais e componentes visuais.

**Infraestrutura e Deploy:**
* [Docker](https://www.docker.com/) - Conteinerização da aplicação.
* [Docker Compose](https://docs.docker.com/compose/) - Orquestração do container e mapeamento de portas/volumes.
